Describe que hacen los siguientes comandos de git, escribe su descripcion al frente en una sola linea.

git status: Este comando se usa para obtener el estado actual del repositorio

git clone: Este comando se utiliza para clonar un repositorio remoto.

git pull: Este comando se utiliza para actualizar los datos del repositorio remoto al local.

git checkout: Este comando se utiliza para cambiar entre ramas de un repositorio.

git log: El comando es utilizado para mostrar un historial del proyecto, filtrarlo y buscar cambios concretos.

git branch: Este comando se utiliza para poder crear ramas en un repositorio de manera local al pushear algo en la rama se va a crear en el repositorio remoto.

git add: Este comando se utiliza para poder agregar los archivos al repositorio de manera indivual o grupal.

git commit: Este comando se utiliza para tener un registro de las acciones realizadas o cambios, para poder movernos entre version de un proyecto. 

git push: Este comando se utiliza para subir los datos del repositorio local al remoto, los archivos o proyectos que tengamos realizados desde el origen que deseamos o la rama elegida.

git merge: Este comando sirve para fucionar las ramas y obtener los archivos de una a otra uniendo de esta manera al igual que facilitar el trabajo.
